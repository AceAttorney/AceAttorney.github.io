Docker部署SmartWiki

1.安装docker

添加docker源

yum安装Docker,启动docker，并添加开机启动

2.部署mysql容器并建立数据库

拉取安装mysql 5.5版本镜像并启动

在启动的容器内建立smartwiki所需数据库

确定下mysql容器内的IP.并记录，启动smartwiki容器时需要使用

3.部署SmartWiki并和mysql容器互联

拉取smartwiki官方镜像并部署

打开防火墙 80端口提供访问

之后就可以使用域名进行访问。

p.s 问题处理

之后可能会发生无法上传图片等问题,原因是smartwiki目录下的storage、public、vendor目录需要777权限。

Grafana-zabbix安装配置

1. 安装Grafana

a. 安装依赖项及mysql

mysql内建立Grafana使用的数据库

b. 下载rpm或者直接使用yum安装

yum安装

rpm安装

安装完成后
配置文件路径为 /etc/grafana/grafana.ini
日志路径为 /var/log/grafana/grafana.log
插件路径为 /var/lib/grafana/plugins/
程序路径为 /usr/sbin/grafana-server
CLI控制台路径为 /usr/sbin/grafana-cli

c. 配置文件使用

修改配置文件使grafana使用mysql
vim /etc/grafana/grafana.ini
修改以下几处为mysql配置

启动Grafana,并添加开机启动。

没有问题后访问测试
http://localhost:3000
默认登录用户名密码为 admin
发现默认可以使用邮箱注册登录，为了防止用户注册，需要修改Grafana配置文件
vim /etc/grafana/grafana.ini

2. 安装Grafana-Zabbix插件

a. 使用grafana-cli或者git安装

使用grafana-cli安装

使用git安装

使用git安装之后可以很方便的升级插件

b. 新建zabbix用户并配置zabbix api

首先在zabbix web界面配置grafana 要使用的用户和密码
先建立一个用户组

编辑用户组权限，让用户组对所有主机有读权限

之后建立用户grafana,并加入之前建立的群组

Grafana登陆后在图形界面配置，点击图示enable now启用zabbix插件

之后点击 Create your first data source,添加zabbix数据源

url填写zabbix实际地址的api

  • Url: set Zabbix API url (full path with api_jsonrpc.php).
  • Access:
    Proxy: access via Grafana backend
    Direct: access from browser.

配置Zabbix API details，填写之前设置的grafana用户及密码，之后保存，grafana会自动测试如果成功连接会提示success，并显示api版本。
这时grafan-zabbix插件就安装完成，可以直接添加各种图形了。
